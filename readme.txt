Add jQuery lazy loading to images

== Description ==

Add lazy loading to your website and that will make your websites faster and your images will be load as after your website get load.

== Installation ==

1. Go to website admin and click add new plugin then simply search with the name of 'jQuery Image Lazy Load WP' click on install and activate.

2. Unzip archive to wp-content/plugins directory, and activate it in Plugins page in wp-admin
